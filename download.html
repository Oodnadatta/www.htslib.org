SAMtools

Downloads

For most users we recommend you download our complete package:
samtools+bcftools+htslib 1.0 (recommended)
Alternatively you can download the individual packages:
samtools 1.0
bcftools 1.0
htslib 1.0

Development

The SAMtools source code is hosted by Github. You can find the project pages here. You can check out the latest development source code with:

			git clone git://github.com/samtools/samtools.git
Please note the source code in this repository is under development and make contain experimental or untested code. We recommend you do not use it in production pipelines unless specifically advised.
